Getting Started

Install, start, and verify Carapace

This page keeps setup on the site: install a binary, launch Carapace locally, run chat, and move directly into recipe-based channel setup.

Download latest release Use the 5-minute recipe

1. Install the binary

Download one artifact from the latest release:

  • cara-x86_64-linux or cara-aarch64-linux
  • cara-x86_64-darwin or cara-aarch64-darwin
  • cara-x86_64-windows.exe

macOS/Linux:

chmod +x ./cara-<your-platform>
sudo mv ./cara-<your-platform> /usr/local/bin/cara
cara --help

Windows (PowerShell):

$installDir = "$env:LOCALAPPDATA\cara\bin"
New-Item -ItemType Directory -Force -Path $installDir | Out-Null
Copy-Item .\cara-x86_64-windows.exe (Join-Path $installDir "cara.exe")
cara --help

2. First run (secure local default)

Generate a service token and run setup:

export CARAPACE_GATEWAY_TOKEN="$(openssl rand -hex 32)"
cara setup
cara

In a second terminal, verify health and open chat:

cara status --host 127.0.0.1 --port 18789
cara chat

In chat, use /help, /new, and /quit.
